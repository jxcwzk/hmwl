# 微信小程序开发调试技能

这是一个专门用于微信小程序开发和调试的技能，帮助你完成小程序的全流程开发工作。

## 何时调用

当用户出现以下情况时，应立即调用此技能：
- 用户需要开发微信小程序
- 用户需要调试微信小程序问题
- 用户需要在小程序中进行接口调试
- 用户需要优化小程序性能
- 用户询问小程序相关问题

## 核心能力

### 1. 代码开发

- 使用微信小程序框架（WXML, WXSS, JS/TS）进行页面和组件开发
- 调用微信小程序原生 API
- 使用小程序开发者工具创建项目
- 配置 app.json 全局配置和页面配置

### 2. 调试能力

- 使用微信开发者工具进行真机预览和模拟器调试
- 查看和分析小程序日志
- 使用 vConsole 进行移动端调试
- 调试页面渲染问题和逻辑错误

### 3. 接口调试

- 使用微信小程序网络请求 API（wx.request）
- 模拟后端接口响应
- 检查请求头、响应体和状态码
- 处理跨域问题和身份验证

### 4. 性能优化

- 分析小程序启动性能
- 检测内存泄漏和性能瓶颈
- 优化页面渲染和setData调用
- 使用性能分析工具监控帧率

## 常用工具

- 微信开发者工具
- 小程序云开发控制台
- Chrome DevTools（真机调试）
- vConsole（移动端调试）

## 项目结构

标准微信小程序项目结构：
```
├── app.js
├── app.json
├── app.wxss
├── pages/
│   └── index/
│       ├── index.js
│       ├── index.json
│       ├── index.wxml
│       └── index.wxss
├── components/
├── utils/
└── images/
```

## 常用命令

```bash
# 使用微信开发者工具命令行
# 预览小程序
wechat-devtools-cli preview

# 构建小程序
wechat-devtools-cli build

# 上传小程序代码
wechat-devtools-cli upload
```

## 注意事项

1. 微信小程序有自己独特的开发语法，不同于普通 Web 开发
2. 小程序运行时环境与浏览器不同，需要注意 API 兼容性
3. 真机调试是必不可少的步骤，模拟器可能存在差异
4. 注意小程序的性能限制，避免内存泄漏和频繁 setData
