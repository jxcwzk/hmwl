### 5.2 网络请求配置
```javascript
// api.js
wx.request({
  url: BASE_URL + options.url,
  method: options.method || 'GET',
  data: options.data || {},
  header: {
    'Content-Type': 'application/json',
    'Authorization': token ? `Bearer ${token}` : '',
    ...options.header
  }
})
```

**验证结果**: ✅ 配置正确，包含认证token支持

### 5.3 发现的问题
⚠️ **小程序API路径不一致**:
- 小程序调用: `/network/list`
- 后端实际路径: `/network-point/list`
- **影响**: 网点列表功能在小程序中无法正常工作
- **建议**: 修改小程序API调用路径或后端添加兼容路由
