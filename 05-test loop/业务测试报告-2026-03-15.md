## 四、问题记录

### 4.1 高优先级

#### 问题1: 数据库未初始化导致无法完整测试
- **描述**: 数据库表不存在，导致 API 调用返回 500 错误
- **错误信息**: `Table 'hmwl.business_customer' doesn't exist`
- **影响**: 无法进行完整的端到端业务测试
- **解决方案**: 执行 `update_db.sql` 和 `init.sql` 初始化数据库
- **状态**: 🔴 待修复

#### 问题2: 敏感凭证配置问题
- **描述**: COS 凭证使用环境变量，但本地开发环境未配置
- **影响**: 图片上传功能无法正常工作
- **解决方案**: 创建 `application-dev.yml` 或配置环境变量
- **状态**: 🔴 待修复

---

### 4.2 中优先级

#### 问题3: 请求拦截器兼容性问题
- **描述**: 部分 API 返回格式不统一（有 code 字段和无 code 字段）
- **影响**: 前端可能解析失败
- **解决方案**: 已在 request.js 中增加兼容性处理
- **状态**: ✅ 已修复

---

### 4.3 低优先级

#### 问题4: 订单状态枚举值
- **描述**: 建议统一订单状态枚举定义
- **影响**: 维护性问题
