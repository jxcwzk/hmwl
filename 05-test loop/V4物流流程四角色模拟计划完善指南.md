## 新增API

### POST /dispatch/confirm-price

客户确认价格接口。

**请求参数:**
```json
{
  "orderId": 123
}
```

**响应:**
```json
{
  "success": true,
  "message": "价格确认成功"
}
```

**状态变更:**
- pricingStatus: 2 → 3
- status: 0 → 5
- priceConfirmedTime: 当前时间
