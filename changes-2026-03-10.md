## 二、调整详情

### 2.1 订单表新增 distance 字段

#### 背景
前端订单详情页面（Order.vue）新增了"距离(km)"字段用于运费计算，但后端 Order 实体类缺少该字段。

#### 后端修改

**文件：** `backend/src/main/java/com/hmwl/entity/Order.java`

```java
// 新增字段
private Double weight;
private Double volume;
private Double distance;  // ✅ 新增
private Double totalFee;
```

#### 数据库修改

**文件：** `backend/src/main/resources/init.sql`

```sql
ALTER TABLE orders ADD COLUMN distance double DEFAULT NULL;
```

#### 影响接口
- `POST /api/order` - 创建订单
- `PUT /api/order` - 更新订单
- `GET /api/order/{id}` - 获取订单详情
- `GET /api/order/page` - 分页查询订单

### 2.2 图片实体拆分

#### 背景
前端订单详情页面新增了两个图片上传卡片：
1. 回单图片（imageType=1）
2. 发货单图片（imageType=2）

原有一个 OrderImage 实体无法清晰区分两种图片类型，故拆分为两个独立实体。

#### 后端修改清单

| 原文件 | 新文件 | 操作 |
|--------|--------|------|
| `OrderImage.java` | `ReceiptImage.java` | 新建 |
| - | `SenderImage.java` | 新建 |
| `OrderImageMapper.java` | `ReceiptImageMapper.java` | 新建 |
| - | `SenderImageMapper.java` | 新建 |
| `OrderImageService.java` | `ReceiptImageService.java` | 新建 |
| - | `SenderImageService.java` | 新建 |
| `OrderImageServiceImpl.java` | `ReceiptImageServiceImpl.java` | 新建 |
| - | `SenderImageServiceImpl.java` | 新建 |
| `OrderImageController.java` | `ReceiptImageController.java` | 新建 |
| - | `SenderImageController.java` | 新建 |

#### 实体类设计

**ReceiptImage.java（回单图片）**
```java
@Data
@TableName("receipt_image")
public class ReceiptImage {
    private Long id;
    private Long orderId;
    private String imageUrl;
    private Integer status;
    private Date createTime;
    private Date updateTime;
}
```

**SenderImage.java（发货单图片）**
```java
@Data
@TableName("sender_image")
public class SenderImage {
    private Long id;
    private Long orderId;
    private String imageUrl;
    private Integer status;
    private Date createTime;
    private Date updateTime;
}
```

#### 新增 API 接口

| 接口路径 | 方法 | 说明 |

| `/api/order/receipt-image/upload/cos` | POST | 上传回单图片 |

| `/api/order/receipt-image/{id}` | GET | 获取回单图片详情 |
| `/api/order/receipt-image/{id}` | DELETE | 删除回单图片 |

| `/api/order/sender-image/list` | GET | 获取发货单图片列表 |

| `/api/order/sender-image/{id}` | DELETE | 删除发货单图片 |

#### 数据库修改

```sql
-- 创建回单图片表
CREATE TABLE IF NOT EXISTS `receipt_image` (
  `id` bigint NOT NULL AUTO_INCREMENT,
  `order_id` bigint DEFAULT NULL,
  `image_url` varchar(1000) DEFAULT NULL,
  `status` int DEFAULT NULL,
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP,
  `update_time` datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  KEY `idx_order_id` (`order_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 创建发货单图片表
CREATE TABLE IF NOT EXISTS `sender_image` (
  `id` bigint NOT NULL AUTO_INCREMENT,
  `order_id` bigint DEFAULT NULL,
  `image_url` varchar(1000) DEFAULT NULL,
  `status` int DEFAULT NULL,
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP,
  `update_time` datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  KEY `idx_order_id` (`order_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```
